# A tiling layout generator for [river](https://github.com/ifreund/river)

## How does it work?

kile creates a dynamic layout of window layouts from an inner and outer layout.

## Building

```shell
git clone https://gitlab.com/snakedye/kile.git
cd kile
cargo build --release
```
